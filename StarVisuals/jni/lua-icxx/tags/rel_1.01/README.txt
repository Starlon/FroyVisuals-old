Lua-icxx v1.01.2011.06
======================

Folders: 

docs	- documentation
src	    - source code
test	- code to test the source
bin	    - where DLL, EXE and LUA scripts get copied by VS
vs-2005-sln - where VS 2005 solution files are located

To build: 

- One environment variable must be defined: LUA_BINARIES, assumed to point to a folder 
  containing lua51.dll and .lib
- Open the solution file in vs-2005-sln and click "build".
